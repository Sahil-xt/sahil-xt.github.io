Case 05 — Field Visit Tracking

Field visits -- transparency fixes trust

Employee-visible tracking + manual check-in/out + expense at checkout

RoleLead UX Designer
PlatformMobile (field) + Web (managers)
UsersSales reps, service techs, managers
Step 1 — Empathise

"Show me the same data you see."

Field employees resisted location tracking — not because tracking was wrong, but because it was opaque. One quote became the north star: "I don't mind being tracked if I can see what they see."

🚗
Field Employee
Sales rep, service tech, driver
Anxiety: Constant surveillance feeling, GPS inaccuracy disputes, no way to correct data
Need: Transparency, location correction option, see own timeline
👔
Field Manager
Supervises team, plans coverage
Problem: Only end-of-day reports, no real-time positions, can't verify visit duration
Need: Live map, visit timeline, territory planning

Key insight: Anxiety wasn't about being tracked — it was about not seeing the same data as the manager. Transparency solved resistance better than any policy change.

Step 2/3 — Define & Ideate

One principle: Employees see exactly what managers see

❌ Passive Tracking (Rejected)
Background GPS constantly running. Employee unaware of exactly what's recorded.
✅ Manual Check-In/Out (Chosen)
Explicit action. Employee awareness. Can correct if GPS is wrong. Same view as manager.

Bonus insight: Capture expense at checkout, not separate. Context is highest at that moment (visit complete, leaving location). Slide-up form instead of separate step later. 3x higher capture rate.
